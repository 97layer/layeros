## I-0. 시각의 침묵 (Visual Silence)

사유를 담아내는 공간의 기본 콘셉트: **"낡은 종이 위에 0.5mm 연필선으로 그린 동심원의 파동"**.

창백하게 낡아가는 회색빛(#E3E2E0) 종이와 먹처럼 단정한 검은 잉크(#1A1A1A)만 허용한다. 덧칠이 아니라 긁어내듯 그린다.

인간의 호흡에 가장 닿아 있는 **3.8초의 맥동**과 절대적으로 강제된 **60% 이상의 텅 빈 여백**. 화면은 화려한 캔버스가 아니라 사유를 담기 위한 빈 그릇이다.

가벼움과 무의미한 유행을 경계한다. 구도자의 도면을 닮은 극단적인 폰트(IBM Plex Mono)만을 건축 자재로 사용한다.

## I-4. Breath System

> 인간의 호흡 주기 ~0.27Hz(3.8초)를 시각적 맥동의 기준으로 삼는다.

| Token | 값 | 의미 |
|-------|-----|------|
| `--breath` | 3.8s | 한 호흡 주기 |
| `--breath-half` | 1.9s | 반 호흡 |
| `--breath-quarter` | 0.95s | 1/4 호흡 |

시간대별 변조: dawn 4.8s / day 3.8s / evening 4.2s / night 5.2s

| Token | 값 | 성격 |
|-------|-----|------|
| `--ease-wave` | cubic-bezier(0.37, 0, 0.63, 1) | 정현파 근사 S곡선 |
| `--ease-out` | cubic-bezier(0.16, 1, 0.3, 1) | 내쉬는 곡선 |
| `--duration` | 0.6s | 기본 전환 |
| `--duration-slow` | 1.2s | 중요 요소 |

## II-1. 에세이 구조

About 페이지 역설계 기반 원칙. CE `write_corpus_essay()` 프롬프트가 SSOT.

- **첫 문장**: 물리적·감각적 관찰로 시작. 원리 선언 금지.
- **비유**: 섹션당 1개. 겹치면 초점 분산.
- **리듬**: 단언(단문) → 탐구(복문) → 열린 결말
- **마무리**: 결론 맺지 않는다. 독자가 다음을 느끼게.
- **전환**: 명시하지 않는다. 이전 섹션의 마지막 이미지가 다음을 불러들임.
- **교훈**: 금지. 관찰만. 독자를 이끌거나 설득하지 않는다.
- **접속사**: 최소화. "또한/그리고/뿐만 아니라" → 단문 분리.
- **수식어**: 1문장 1개 이하.

### II-1.5. 언어 감도 세부 규칙

> 순호의 미세조정에서 귀납한 알고리즘. 세션마다 갱신.
> (이전 글쓰기 가이드 흡수 — 2026-03-04)

#### 어휘 교체

| 금지 | 이유 | 대안 |
|------|------|------|
| `나다운가`, `나다움` | K-웰니스 클리셰 | `조용히 돌아오는가` |
| `신호` (내면 상태) | 기계적 뉘앙스 | `감각` |
| `부정할 수 없는 X` | 수식 과잉, 선언력 약화 | 수식어 삭제, 단언으로 |
| `~함에 있어서`, `이러한 ~에서` | 번역투 | 단문 분리 |
| `또한`, `그리고`, `뿐만 아니라` | 접속사 과잉 | 문장 분리, 침묵 연결 |
| `—` (em dash) | 과잉 강조, 순호 명시 금지 | 마침표 또는 쉼표 |
| `아름다운`, `ethereal`, `dreamy` | AI 과잉 수식 | 물리적 사실, 긴장 |
| `이것이 X의 이유입니다` | 과설명, 독자 여백 침범 | 이전 문장이 이유. 설명 불필요 |

#### 감각 언어 vs 기계 언어

| 기계 언어 ❌ | 감각 언어 ✅ |
|------------|------------|
| `신호` | `감각` |
| `방향성` | `방향` |
| `시스템`, `메커니즘`, `프로세스` | — (삭제) |

#### 감정 공명 메커니즘

| 패턴 | 작동 원리 |
|------|---------|
| **미표현 언어화** | 느꼈지만 말한 적 없던 감각을 정확히 포착 |
| **과거형 균열** | 현재의 나 vs 과거의 나 균열 전제 |
| **기반 흔들림** | 당연한 것에 의문 제기 — 거대 신념보다 그 아래가 더 깊은 충격 |
| **취약성 고백** | 화자/브랜드가 먼저 "모른다" 인정 → 독자도 취약해질 허락 |
| **불완전 수용** | "완성이 목적이 아닙니다" → 위로가 아닌 구조적 허락 |

#### 세션별 누적 교정 (최신순)

- 2026-03-03: `신호` → `감각` (내면 상태) / `나다운가` → `조용히 돌아오는가` / `자기 자신이 누구인지` → `어디를 향하는지` / Das Man 감도 "익숙한 것, 편한 것, 무난한 것" 3개 고정, 확장 금지
