# WOOHWAHAE 산문 언어 감도 — Writing Guide

> 순호의 미세조정에서 귀납한 알고리즘. 세션마다 갱신.
> 마지막 갱신: 2026-03-03

---

## 1. 어휘 교체 규칙

### ❌ 금지 어휘 (클리셰 / 감도 오류)

| 금지 | 이유 | 대안 |
|------|------|------|
| `나다운가`, `나다움` | K-웰니스 브랜드 공통 클리셰 | `조용히 돌아오는가`, `제 것이라 느껴지는가` |
| `신호` (내면 상태 표현 시) | 기계적·기술적 뉘앙스 | `감각` |
| `부정할 수 없는 X` | 불필요한 수식. 선언력 약화 | 수식어 삭제, 단언으로 |
| `~함에 있어서`, `이러한 ~에서` | 번역투 | 단문으로 쪼갬 |
| `또한`, `그리고`, `뿐만 아니라` | 접속사 과잉 | 문장 분리, 침묵으로 연결 |
| `—` (em dash) | 과잉 강조, 순호가 명시적으로 금지 | 마침표 또는 쉼표로 대체 |
| `첫 번째 X` (같은 문장 내 X 반복) | 중복 | 삭제 또는 단어 교체 |
| `아름다운`, `ethereal`, `dreamy` | AI 과잉 수식 | 물리적 사실, 관계, 긴장 |
| `이것이 X의 이유입니다` | 과설명. 독자 여백 침범 | 이전 문장이 이유. 설명 불필요 |

## 2. 문장 구조 원칙

### 리듬
- **短-長-短**: 선언 → 탐구 → 여운
- 한 문단 2–4문장. 5문장 이상이면 밀도 분산
- 철학 원리: `~이다` / `~한다` (단언)
- 해석 여지: `~라고 본다` / `~처럼 보인다`

### 비유
- **섹션당 1개**. 겹치면 초점 분산
- 현재 배정된 비유:
  - 서(序): 없음 (직접 질문)
  - 소거: 조각가
  - 느림: 익지 않은 것 / 고치
  - 무언의교감: 소리굽쇠
  - 우화해: 유충 → 나방 (어원에 내재)

### 마무리
- **각 섹션 마지막 문장은 열려있어야 한다** — 결론 맺지 않음
- 독자가 다음 섹션을 느끼게. 전환을 설명하지 않음

| 기계 언어 (❌) | 감각 언어 (✅) |
|--------------|--------------|
| `신호` | `감각` |
| `시스템` | — |
| `메커니즘` | — |
| `프로세스` | — |
| `방향성` | `방향` |

## 4. 철학적 뿌리 — 구조로 사용, 명시는 최소

현재 about에 작동 중인 개념들:

| 개념 | 철학 뿌리 | 텍스트 내 역할 |
|------|---------|-------------|
| 소거 | 하이데거 알레테이아(ἀλήθεια) — 비은폐성 | 핵심 원리 |
| 외부 기준 내면화 | 하이데거 Das Man(세인) | 서(序) P2 "누구도 탓하지 않을 것" |
| 영점 | 현상학 에포케(epoché) | 소거의 도착점 |
| 공명 | 물리학 공진 + 융 싱크로니시티 | 무언의 교감 |
| 본래면목 | 선불교 + 하이데거 본래성(Eigentlichkeit) | 자기긍정 |
| 소우주 | 天人合一(천인합일) | 발(跋) 잠재 삽입 지점 |

> 명시적 인용은 1–2개 이하. 나머지는 구조로 작동시킨다.

## 5. 감정 공명 메커니즘

> 독자를 울컥하게 만드는 구조적 패턴. 의도적으로 설계 가능.

| 패턴 | 작동 원리 | 예시 |
|------|---------|------|
| **미표현 언어화** | 느꼈지만 말한 적 없던 감각을 정확하게 포착 | "좋아한다고 생각했던 것을 실제로 좋아하는지 모르겠다는 것" |
| **과거형 균열** | 현재의 나 vs 과거의 나 사이 균열 전제 | "~이라고 생각했던 것이 어느 순간 낯설어진다" |
| **기반 흔들림** | 당연한 것(좋아함/싫어함)에 의문 제기 | 거대 신념이 아니라 그 아래가 흔들릴 때 충격이 더 깊다 |
| **취약성 고백** | 화자/브랜드가 먼저 "모른다" 인정 | 독자도 취약해질 허락을 받는다 |
| **불완전 수용** | 완성되지 않아도 된다는 구조적 허락 | "대답은 완성되지 않습니다" — 위로가 아닌 허락 |

**불완전 수용 vs 웰니스 위로:**
- 웰니스: "당신은 충분합니다" → 위로, 외부에서 주어짐
- 불완전 수용: "완성이 목적이 아닙니다" → 철학 안에서 나오는 허락, 구조적

## 6. 세션별 추가 기록

### 2026-03-03
- `신호` → `감각` (내면 상태 표현 시)
- `나다운가` → `조용히 돌아오는가`
- `자기 자신이 누구인지` → `자기 자신이 어디를 향하는지`
- 005 서브레이블: 形(형)/無形(무형)/本(본) 한글 병기 확정
- 발(跋) `공간, 언어, 기록` 열거 삭제 — 구 레이블 흔적
- Das Man 감도: "익숙한 것, 편한 것, 무난한 것" — 이 3개 리스트 그대로 사용. 확장 금지.
